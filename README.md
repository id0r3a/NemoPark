# Parking Service 
This project implements a parking service system using C# and the Spectre.Console library. It provides functionality to start and end parking sessions for vehicles, save and load parking data from a JSON file, and display a summary of all parking sessions. 

## Features 
- Start and end parking sessions
- Save parking data to a JSON file
- Load parking data from a JSON file
- Display all parking sessions in a table format 
- Calculate parking costs based on duration
